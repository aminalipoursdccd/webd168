Understanding the CSS Box Model

Every HTML element is treated like a box. The box model helps control spacing, size, and layout by using content, padding, border, and margin.

Content

The content is the text or media inside an element. It is the main part of the box.

Padding & Border

Padding adds space inside the element, while the border creates the visible edge around it.

Margin

Margin adds space outside the element, separating it from other boxes on the page.
